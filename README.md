# Angola Finance

A sua plataforma de finanças inteligentes em Angola.

## Projecto
Este projecto é uma aplicação de gestão financeira adaptada à realidade angolana, com ferramentas de orçamento, poupança, investimentos e educação financeira.

## Tecnologias
- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
- Supabase

## Instalação

```sh
# Instalar dependências
npm i

# Iniciar servidor de desenvolvimento
npm run dev
```
